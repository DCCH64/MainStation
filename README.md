# MainStation

> This is a work in progress! Please don't expect much from it now in this very early state.

## Installation

For the ease of installation and maximum compatibility, we chose to offer you prebuilt docker images to install on your OS of choice. You can find some auto-built docker images [here](https://hub.docker.com/r/erikd256/dcch64-mainstation). Even if we try to make this software as generic as possible, we can not guarantee that this piece of software works on every SBC without **(minor or major)** modifications.
